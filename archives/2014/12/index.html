Dec 3 2014

hexo-use-note

执行sudo hexo d 报错

Permission denied (publickey).
fatal: The remote end hung up unexpectedly

早上刚到公司，解决了这个问题
原因是sudo会调用root用户的ssh key

sudo chown -R user.user hexo
hexo d # 执行成功！
